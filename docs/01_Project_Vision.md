# Mehmaan Hub - Project Vision

## Project Overview

Mehmaan Hub is a modern rental platform built to simplify the process of finding and listing rental properties across Pakistan.

The platform connects property owners with students, working professionals, and families looking for rooms, hostels, apartments, or houses.

Our goal is to provide a secure, trusted, and user-friendly rental experience.

---

# Mission

To make renting and listing properties simple, trustworthy, and fast.

---

# Vision

To become Pakistan's most trusted rental platform where anyone can easily find or list rental properties.

---

# Problem Statement

Currently, people search for rental properties using Facebook groups, WhatsApp groups, or local dealers.

These methods have several problems:

- Fake property listings
- No verification
- Poor search experience
- No proper communication
- Difficult to compare properties

Mehmaan Hub solves these problems by providing one trusted platform.
